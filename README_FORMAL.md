# Copy-Paste Augmentation Project

A computer vision research project exploring copy-paste augmentation in a structured and reproducible way.

## Overview
This repository studies how copy-paste style data processing can be used in a model training workflow. The project combines dataset preparation, experiment execution, and result comparison.

## Goals
- Build a reusable experiment workflow
- Compare baseline and processed-dataset runs
- Document findings, trade-offs, and follow-up ideas
- Keep notebook and script workflows understandable

## Main Areas
- Data preparation
- Augmentation workflow
- Training runs
- Result comparison
- Research notes

## Dependencies
The repository currently references packages such as Detectron2, NumPy, scikit-learn, shapely, pyclipper, matplotlib, and OpenCV.

## Recommended Structure
```text
data/
notebooks/
scripts/
results/
docs/
```

## Getting Started
1. Review setup notes and dependency definitions
2. Prepare data and annotations
3. Run the script or notebook workflow
4. Compare outputs across runs
